# PySpark Data Processing Assignment

## Overview

This project demonstrates data processing and analytics using Apache Spark and PySpark on the Superstore dataset.

The assignment includes:

* Data loading
* Data cleaning
* Schema modification
* Filtering operations
* Aggregations and groupBy
* Wide transformations
* SQL operations
* Complete data processing pipeline
* Business insights generation

## Technologies Used

* Python
* PySpark
* Apache Spark
* Jupyter Notebook

## Dataset

Superstore Sales Dataset

## Key Operations Performed

* Null value handling
* Duplicate removal
* Column transformations
* Data filtering
* Aggregations
* SQL queries
* End-to-end Spark pipeline implementation

## Conclusion

This assignment demonstrates the use of PySpark for scalable data processing and analytical operations on structured datasets.
